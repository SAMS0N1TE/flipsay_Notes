FlipSay
◈ PUBLIC BETA ◈
Welcome to FlipSay — the first browser-based
SDR-style interface for Flipper Zero.

Connect your Flipper via USB and start
exploring the radio spectrum in real time.
  • Requires Momentum firmware (developer version)
  • Enable Debug Mode on your Flipper first
  • Use Google Chrome or Microsoft Edge
  • Close qFlipper before connecting
  • This is a beta — bugs may occur. Report them on GitHub!
